Getting Started

1

Sign Up or Login

Create a new account or login with your existing credentials to access the system.

2

Navigate the Menu

Use the main menu to access different features like adding students, viewing reports, and managing scores.

Managing Students

  • Add Student: Enter student details including name, ID, class, and subjects
  • Edit Student: Update student information and subject details
  • Delete Student: Remove students from the system
  • Search Students: Find students by name, ID, or class

Score Management

  • Enter Scores: Input test scores, CBT, and theory exam marks for each subject
  • Automatic Calculation: System automatically calculates totals and grades
  • Grade Boundaries: A (70+), B (60-69), C (50-59), D (40-49), F (below 40)
  • Validation: System validates score ranges and prevents invalid entries

Reports & Analytics

  • View Students: See all students with their scores and grades
  • Student Reports: Detailed individual student performance reports
  • Class Reports: Overview of entire class performance
  • Ranking System: Automatic ranking based on average marks

Ranking & Positions

The system provides automatic ranking based on average marks:

  • Students are ranked within their class
  • Higher average marks get better positions
  • Tie-breaking is handled automatically
  • Positions update when scores change

Reporting Issues

  • Use the "Report Issue" feature to submit technical problems
  • Provide detailed descriptions of issues encountered
  • Issues are reviewed by administrators
  • You'll receive updates on reported issues

Frequently Asked Questions

How do I add multiple subjects for a student?
When adding a student, enter the number of subjects first, then list all subjects separated by commas in the subjects field (e.g., "Mathematics, English, Physics").
What score ranges are valid?
First Test, Second Test, Third Test: 0-10 each
CBT/Objective: 0-30
Theory Exam: 0-40
The system will validate and prevent invalid score entries.
How are grades calculated?
Grades are based on overall marks (tests + exams):
A: 70-100, B: 60-69, C: 50-59, D: 40-49, F: 0-39
Pass/Fail status: Pass (60+), Fail (below 60)
Can I export student data?
Yes, you can export student reports and rankings in various formats. Look for export options in the reports section.
How do I reset a student's scores?
Go to "Enter Scores" for the student and submit new scores. The system will automatically update and recalculate grades.
